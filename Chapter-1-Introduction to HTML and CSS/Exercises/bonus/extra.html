Some of HTML Codes.

Basic HTML Tags We Learned

<h1> to <h6> - Headings

Used to define headings, from largest (<h1>) to smallest (<h6>).

<h1>Main Heading</h1>
<h2>Subheading</h2>

<p> - Paragraph

Defines a block of text.

<p>This is a paragraph.</p>

<figure> and <figcaption> - Multimedia Content

Used for self-contained content, like images with captions.

<figure>
<img src="image.jpg" alt="Description" />
<figcaption>Caption text here</figcaption>
</figure>
